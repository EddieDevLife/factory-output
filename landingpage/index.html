Conecte com seu ecossistema de dados

O Agentix Vault publica métricas em JSON e CSV via GitHub Pages. Pronto para consumir no Tableau, Databricks ou Deepnote.

Ou faça ingest em Postgres (gratuito no Neon) e monte seus próprios dashboards.

Arquitetura pensada para confiança

  • 🔒 CSP e Headers — Content Security Policy restritiva e headers de segurança por padrão.
  • 🐢 Rate Limiting — Proteção contra abusos na API de monitoramento.
  • 📋 Logs estruturados — Tudo em JSON para auditoria e debug eficiente.
  • 🔑 Menor privilégio — Cada módulo (Hive, Pulse, Vault) opera com o mínimo de acesso necessário.

Perguntas frequentes

Preciso hospedar um backend?

Não. O Vault publica métricas como arquivos estáticos (GitHub Pages). Você consome os dados direto via URL em qualquer ferramenta de BI.

Como publico entregas e métricas?

O pipeline (GitHub Actions) processa o conteúdo em entregas/ e publica os artefatos em site/, incluindo deliveries_index.json e runs.csv.

Isso funciona localmente?

Sim. Você pode rodar o exportador localmente para gerar site/ e testar a landing page apontando para o arquivo gerado.

Onde encontro os arquivos prontos?

No GitHub Pages do repositório: site/deliveries_index.json (JSON) e site/runs.csv (CSV). Você pode linkar direto do seu dashboard.

Como evoluir para produção?

Deixe o Vault publicando via GitHub Pages e, se precisar de histórico/BI avançado, habilite ingest em Postgres (Neon). Para observabilidade em tempo real, conecte um dashboard (Pulse) consumindo os endpoints do Vault.
